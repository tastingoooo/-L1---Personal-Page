# YI CEHN CEHN - Personal Page

A modern, dynamic, and responsive personal web page.

## Features

- **Sleek & Modern Design**: Dark theme with high-contrast text and subtle gradients.
- **Responsive Web Design (RWD)**: Adjusts smoothly to mobile phones, tablets, and desktop screens for an optimal viewing experience.
- **Dynamic Clock widget**: A live-updating local time display directly integrated into the hero section, utilizing JavaScript.
- **Glassmorphism Elements**: Beautiful semi-transparent surfaces with background blur.
- **Subtle Animations**: Smooth fade-in effects and interactive hover states.

## Technologies Used

- **HTML5**: For semantic structuring.
- **CSS3**: For styling, animations, flexbox layouts, and custom properties (variables), without needing external bulky frameworks.
- **Vanilla JavaScript**: Used strictly for the real-time clock updating mechanism.

## Setup / Installation

Since this is a static webpage built with standard web technologies, there's no build process required to run it locally.

1. Clone the repository:
   ```bash
   git clone https://github.com/tastingoooo/-L1---Personal-Page.git
   ```
2. Open `index.html` in your favorite modern web browser.

## Deployment

This repository is set up with a `gh-pages` branch, making it extremely easy to host on GitHub Pages. If the site is not visible, ensure GitHub Pages is enabled in your repository's settings pointing to the `gh-pages` branch.
